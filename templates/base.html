Frequently Ask Questions

What is Sentiment Analysis?

Sentiment analysis is a process which extracts information from written texts and aims to determine the attitude of an individual.

How does it work?

The tweets from the twitter handle inputted are retrieved and passed through an algorithm that classifies the tweets into positive, negative or neutral tweets.

How long does it take to run the analysis?

It takes approximately one minute and thirty seconds to run the analysis

How far back can I go when selecting the duration?

You can only run analysis on tweets that are less than a week old.

How much does it cost?

It's free. It's an open source project.
